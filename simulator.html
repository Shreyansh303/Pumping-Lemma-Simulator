FORMAL DEFINITION

For any regular language L, there exists a pumping length p such that any string s ∈ L with |s| ≥ p can be split into s = xyz with:

  • 0. |s| ≥ p (Initial Condition)
  • 1. |xy| ≤ p
  • 2. |y| > 0
  • 3. For all i ≥ 0, xyⁱz ∈ L
